## 🛠 How It Works (Simple Explanation)

Rust runs your program without garbage collection (GC). This means it manages memory directly by tracking who owns what and when to free resources.

ferroscope shows this process visually. It tracks how much memory your program uses, checks how fast it runs, and reveals Rust's smart memory rules live.

This helps you understand key Rust ideas like:

- Ownership: who controls each piece of memory  
- Borrowing: accessing data without copying  
- Zero-cost abstractions: getting smart features without slowing down
